Welcome to the World of HTML & CSS!

Introduction

What is HTML?

HTML is the skeleton of a webpage. It holds text, images, videos etc.

What is CSS?

Styles the webpage to make it more appealing.

HTML Basics

What is a tag?

A tag signifies how an element is represented on the web page.

What is an element?

The element is the core item that is visible to anyone who visits the web page.

What is an attribute?

An attributes adds information and values to elements. It can be used for Css or JavaScript.

CSS Basics

What is a selector?

A selector is a declaration that identifies which elements are to be styled.

What is a class?

A class is an attribute that can be implemented in HTML and can group multiple elements together for styling purposes.

What is an ID?

An ID is an attribute that can only be implemented once per element for styling purposes
